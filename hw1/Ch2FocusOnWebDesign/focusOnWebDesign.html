Website 2: St. Louis Church Website

What is the URL of the website?
St. Louis Church

Is the page appealing or unappealing? List three reasons for your answer.
  1. I think this page is unappealing because the overall look and layout of the page feels very outdated and old. Everything is seperated into blocks and the website does not feel modern at all.
  2. The page would be more appealing if it had more animated parts, like a carousel or fading pictures as you scroll.
  3. I like pages with a lot of information on the front page, but I think there is too much information on this front page which makes to too busy.
If the page is unappealing, what would you do to improve it?
To improve this page, I would focus on making it look more modern. I would remove some of the blocks on the main page that can be removed or put on a different page and I would add moving, dynamic elements like photos and videos that appear as you scroll. I would also make the font size bigger where possible and change the font in some parts.
