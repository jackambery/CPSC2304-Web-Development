Network Protocols

  • FTP - File Transfer Protocol is a set of rules that allow files to be exchanged between computers on the Internet.
    Read more about FTP here.
  • HTTP - Hypertext Transfer Protocol is a set of rules for enchanging files such as text, images, audio, video, and other multimedia on the Web.
    Read more about HTTP here.
  • TCP/IP - Transmission Control Protocol/Internet Protocol has been adopted as the official communication protocol of the Internet. The purpose of TCP is to ensure the integrity of network communication. IP works with TCP. It is a set of rules that controls how data is sent between devices on the Internet.
    Read More about TCP/IP here.
